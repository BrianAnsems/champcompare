This project will allow for the user to search for two League of Legends champions. Using Riot's API documentation, the champions' data are displayed within the table below. The higher stat is made bold, and the absolute differences between the stats are displayed in the final column.

I started this project to learn how to parse JSON files and create HTML tables.

Future features may include itemization and other customization available within the game.
